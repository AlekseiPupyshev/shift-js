Basic Declarations and Assignment

Variables and constants will be converted to their JS equivalent. Type annotations are ignored, and the let keyword is converted to var. There is no support for the various number or character types in Swift. Optionals are not supported. Semi-colons will be inserted at the end of each line of JS, regardless of the Swift input. 

  var name: String = "Joe"
  let num: Int = 5;
  var anotherNum = 6
  var yetAnotherNum: Float = 4.2;
  let truth: Bool = false
  var emptyString: String
  var first, second, third: String
  var name = 'Joe';
  var num = 5;
  var anotherNum = 6;
  var yetAnotherNum = 4.2;
  var truth = false;
  var emptyString;
  var first, second, third;

Dictionaries and Arrays

Swift arrays are converted to JS arrays, while dictionaries are converted to object literals. Values are accessed via subscript lookup. There is comprehensive support for anested collections. ShiftJS does not yet support sets. 

  var e = ["Eggs", "Milk", "Bacon"];
  var f = e[0]
  var g = ["one": 1, "two": 2, "three": 3];
  var h = g["one"]
  let i = [
    "array1": [1,2,3],
    "array2": [4,5,6]
  ]
  var e = ["Eggs", "Milk", "Bacon"];
  var f = e[0];
  var g = {'one': 1, 'two': 2, 'three': 3};
  var h = g['one']
  var i = {
    'array1': [1, 2, 3],
    'array2': [4, 5, 6]
  };

String Interpolation and Concatenation

There is full support for string concatenation and interpolation.

  var name = "Sam";
  var n = "Hi my name is " + name + ".";
  var planet = "Earth";
  let o = "Hello \(planet)!";
  var p = "\(100 - 99), 2, 3"
  var name = "Sam";
  var n = "Hi my name is " + name + ".";
  var planet = 'Earth';
  var o = 'Hello ' + planet + '!';
  var p = '' + 100 - 99 + ', 2, 3';

Tuples

Tuples are converted to object literals. If element names are not provided, they are assigned numeric keys, indexed from zero, based on their order in the tuple. There is no support yet for tuples containing nested collections 

  var error = (404, "not found")
  let http = (status: 200, desc: "OK")
  var error = {0: 404, 1: 'not found'};
  var http = {status: 200, desc: 'OK'};

Mathematical and Logical Operators

There is full support for mathematical and logical operators. For math operations, parentheses will be included in the JS output only if they have significance for the order of operations. 

  var answer = (5 * 6) / 4 - (-1 + 4.2) % 2
  var a = 1; var m = ++a; var n = --m;
  var answer = 5 * 6 / 4 - (-1 + 4.2) % 2;
  var a = 1; var m = ++a; var n = --m;

Control Flow

Tuples are converted to object literals. If element names are not provided, they are assigned numeric keys, indexed from zero, based on their order in the tuple. There is no support yet for tuples containing nested collections 

  var gameInProgress = false;
  var score = 0;
  var typeOfScore = ""
  var PAT = "";
  while gameInProgress {
    if (typeOfScore != "") {
      if typeOfScore == "TD" {
        score += 6
      } else if typeOfScore == "PAT" {
        if PAT == "TD" {
          score += 2
        } else {
          score += 1
        }
      } else if (typeOfScore == "FG") {
        score += 3
      } else {
        score += 2;
      }
      typeOfScore = ""
    }
  }
  var gameInProgress = false;
  var score = 0;
  var typeOfScore = '';
  var PAT = '';
  while (gameInProgress) {
    if (typeOfScore != '') {
      if (typeOfScore == 'TD') {
        score += 6;
      } else if (typeOfScore == 'PAT') {
        if (PAT == 'TD') {
          score += 2;
        } else {
          score += 1;
        }
      } else if (typeOfScore == 'FG') {
        score += 3;
      } else {
        score += 2;
      }
      typeOfScore = '';
    }
  }

For-In Loops

There is full support for-in loops

  let interestingNumbers = [
    "Prime": [2, 3, 5, 7, 11, 13],
    "Fibonacci": [1, 1, 2, 3, 5, 8],
    "Square": [1, 4, 9, 16, 25],
  ]
  var largest = 0
  for (kind, numbers) in interestingNumbers {
    for number in numbers {
      if number > largest {
        largest = number
      }
    }
  }
  var interestingNumbers = {
    'Prime': [2, 3, 5, 7, 11, 13],
    'Fibonacci': [1, 1, 2, 3, 5, 8],
    'Square': [1, 4, 9, 16, 25]
  };
  var largest = 0;
  for (var kind in interestingNumbers) {
    var numbers = interestingNumbers[kind];
    for (var number in numbers) {
      if (number > largest) {
        largest = number;
      }
    }
  }

Functions

The transpiler has very broad support for most function patterns, including functions that take a function as an argument or return a function, and functions with variadic parameters. 

  func greet(name: String) -> String {
    return "Hello \(name)."
  }
  greet(name: "Bob")

  func newIncrementer() -> ((Int) -> Int) {
    func addOne(number: Int) -> Int {
      func anon(n: Int) -> Int {
        return 1 + n
      }
      return anon(number)
    }
    return addOne
  }
  function greet(name) {
    return 'Hello ' + name + '.';
  }
  greet('Bob');

  function newIncrementer() {
    function addOne(number) {
      function anon(n) {
        return 1 + n;
      }
      return anon(number);
    }
    return addOne;
  }
